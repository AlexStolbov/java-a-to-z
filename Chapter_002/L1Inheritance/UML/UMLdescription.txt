Описание классов.

Item - Общий класс заявка. Содержит общие поля и методы для всех заявок. От него наследуются классы для различных видов заявок.
	Поля:
	    id - уникальный номер заявки.
	    name - имя заявки
	    description - описание заявки
	    comments -  массив комментариев содержит объекты типа Comment. Т.е. одна заявка может содержать много комментариев
	    createDate - дата создания заявки
	Методы:
	    Item(String id, String name) - конструктор, создает объект по заданным id и name.
	    getId() - геттер id.
	    getName() - геттер name.
	    getComment() - геттер comments.
	    addComment(Comment comment) - добавляет новый комментарий в заявку

Наследники Item. Реализуют особенности различных видов заявок. Предполагается, что разные виды заявок нужно будет обрабатыавть по разному.
Task - заявка задача.
Bug - заявка об ошибке.

Comment - класс комментарий к заявке. Комментарий со временем может обрасти доп. полями и методами, поэтому в отдельный класс.
    Поля:
        id - уникальный номер комментария
        commentText - текст комментария
    Методы:
        Comment(String id, String CommentText) - конструктор

Tracker - класс, содержащий все заявки и обеспечивающий работу с ними, как со множеством. 
	Поля:
        items - массив всех заявок, объекты типа Item
        Random - объект используется в generateId().
    Методы:
        addItem(Item item) - Добавляет новую заявку.
        generateId() - генерирует случайный id для заявок и комментариев.
        editItem(Item item, String name, String descr) - редактирование заявки, переданной заявке устанавливается имя и описание.
        deleteItem(Item item) - Удаляет заявку,
        getListAllItems() - Возвращает массив всех заявок.
        getListFilteredItems(String name) - Возвращает массив заявок, отфильтрованных по вхождению строки в наименование.
        addItemsComment(Item item, String commentText) - добавляет комментарий к заявке.

Display - Запускающий класс. Визуализация и взаимодействие с пользователем. Отображает необходимую информацию, обрабатыает ввод от пользователя.
	Поля:
        tracker - объект класса Tracker, который визуализируем.
    Методы:
        main() - старт программы
        displayListItems() - показать список заявок
        getActiveFromUsers() - показать меню действий, получить выбор пользователя.
